# Basic Calculator

Built using HTML, CSS, and JavaScript

## Description

A responsive and user-friendly calculator web application that performs basic mathematical operations.

This project includes custom expression parsing logic with regular expression validation, operator precedence handling, unary minus support, and decimal number support.

The calculator does not use `eval()`. Expressions are parsed and evaluated using custom JavaScript logic.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Power (Exponentiation)
- Decimal number support
- Unary minus handling (e.g., -5+3, 8Ã—-2)
- Regex-based invalid expression detection
- Proper operator precedence implementation
- Responsive UI
- Clickable buttons for operands and operators

## Expression Handling

The calculator includes:

- Validation of invalid operator sequences
- Prevention of invalid decimal formats
- Detection of invalid starting or ending operators
- Regex-based parsing of numbers including optional unary minus
- Separate operand and operator arrays for controlled evaluation
